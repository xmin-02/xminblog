본 블로그는 로컬 환경에서 실시되었으며, wsl과 로컬 환경 동일합니다.

linux-6.17.6 버전을 기준으로 작성 되었으며, 모든 설치 과정은 동일하니 linux-6.17.6 부분만 각자 다운받은 버전으로 변경하면 됩니다.

본 블로그의 작업 디렉토리는 /usr/src/kernel 입니다.

## Step 1: 커널 소스 다운로드

https://kernel.org/ 에서 다운로드

## Step 2: 초기 설정

```bash
sudo mkdir /usr/src/kernel
cd /usr/src/kernel
sudo cp ~/Downloads/linux-linux-6.17.6.tar.xz ./
```

## Step 3: 압축 해제 및 준비

```bash
sudo su
tar -xvf linux-6.17.6.tar.xz
```

## Step 4: 의존성 설치

```bash
sudo apt update
sudo apt install -y build-essential libncurses-dev bison flex libssl-dev libelf-dev dwarves libdw-dev libdwarf-dev zlib1g-dev libiberty-dev bc fakeroot cpio kmod rsync curl clang llvm lld gawk
```

## Step 5: 커널 옵션 설정

퍼징 관련 설정을 활성화합니다:

- Security signing 비활성화
- KCOV (coverage collection)
- KASAN (heap overflow detection)
- KMSAN (uninitialized memory detection)
- UBSAN (undefined behavior detection)
- Lockdep (lock debugging)
- Debug information 설정

## Step 6: 커널 빌드

```bash
make -j$(nproc)
```

빌드 시간은 시스템 리소스에 따라 10분에서 1시간 정도 소요됩니다.

## Step 7: 모듈 설치

```bash
make modules
sudo make modules_install
sudo make install
```

"Kernel: arch/x86/boot/bzImage is ready" 출력과 모듈 설치 완료로 성공을 확인할 수 있습니다. 빌드된 커널은 `/boot/vmlinuz-6.17.6`에 위치합니다.
