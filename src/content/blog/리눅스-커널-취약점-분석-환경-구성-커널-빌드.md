# [리눅스 커널 취약점 분석] 환경 구성, 커널 빌드

## 환경

- VMware Workstation Pro 17
- Ubuntu_24.04.3_LTS

## 하드웨어 구성

![하드웨어 구성](https://blog.kakaocdn.net/dna/qNMjK/btsP627z5yc/AAAAAAAAAAAAAAAAAAAAANOhSbNV7pNKT-2z9CX5_BX4yj5FRNuAaXJx1I3M25_I/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1777561199&allow_ip=&allow_referer=&signature=0TGyrliuBb%2FqAbTt%2FqSDngMt%2Fhw%3D)

## VM Tools 설치

```bash
sudo apt-get autoremove open-vm-tools
sudo apt-get install open-vm-tools-desktop
```

## 커널 소스 다운로드

```bash
wget https://cdn.kernel.org/pub/linux/kernel/v6.x/linux-6.13.2.tar.xz
tar -xvf linux-6.13.2.tar.xz
cd linux-6.13.2
```

## 빌드 의존성 설치

```bash
sudo apt-get install -y build-essential libncurses-dev bison flex libssl-dev libelf-dev dwarves libdw-dev libdwarf-dev zlib1g-dev libiberty-dev bc fakeroot cpio kmod rsync curl clang llvm lld gawk
```

## 커널 설정

```bash
uname -a
sudo cp /boot/config-6.14.0-28-generic ./.config
make menuconfig
```

![menuconfig 화면](https://blog.kakaocdn.net/dna/bm1T5Q/btsP47I1WjE/AAAAAAAAAAAAAAAAAAAAAEzdELwoFVjqONUeF0OqLzBWXerjbGkmzWpGNMSIerzL/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1777561199&allow_ip=&allow_referer=&signature=H0eAzvi4qxAlRj7RiSfU6B8fSiI%3D)

## 보안 키 설정 제거

```bash
scripts/config --set-str SYSTEM_TRUSTED_KEYS ""
scripts/config --set-str SYSTEM_REVOCATION_KEYS ""
scripts/config --set-str SYSTEM_BLACKLIST_HASH_LIST ""
```

## 커널 컴파일

```bash
make mrproper
make olddefconfig
make -j"$(nproc)"
```

## GRUB 설정

```bash
sudo vi /etc/default/grub
GRUB_TIMEOUT_STYLE=menu
GRUB_TIMEOUT=10
GRUB_TERMINAL=console
sudo update-grub
```
